# Retrieval Knowledge Base (SBERT + FAISS)

This folder (`Data/knowledge/`) is the set of .txt files that we index in FAISS.

Files:
- fitness.txt
- mutation_rules.txt
- pacman_strategies.txt
- heuristics.txt
- code_templates.txt

Note:
- `knowledge_docs/` is project documentation (useful for report/slides) and does not have to be indexed.
