## How to Run (commands)
From the repo root:

```bash
python -m venv .venv
# Windows CMD:
.venv\Scripts\activate.bat
pip install -U pip
pip install -e .
streamlit run UI/AlgoChat.py

```
## Flow of Execution (high-level)
1) User enters algorithm description (optional) and initial code (required).
2) User sets fitness weights and number of generations.
3) System runs evolution for N generations (generate -> evaluate -> select).
4) UI shows final best fitness/score and saves CSV output.

## Issues experienced + solutions
- PowerShell could not activate venv (scripts disabled).
  - Solution: use CMD activation (`.venv\Scripts\activate.bat`) or run `.venv\Scripts\python ...`
- Streamlit missing: `No module named streamlit`
  - Solution: `pip install streamlit`
